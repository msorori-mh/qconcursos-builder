

# خطة إضافة محاكاة التجارب المعملية (PhET) إلى المنصة

## الفكرة العامة

إضافة تبويب "تجربة معملية" داخل صفحة الدرس يعرض محاكيات PhET التفاعلية عبر iframe. يتم ربط كل درس بمحاكاة أو أكثر من خلال جدول جديد في قاعدة البيانات، ويتحكم الأدمن بإضافتها من لوحة التحكم.

PhET توفر محاكيات مجانية مفتوحة المصدر يمكن تضمينها مباشرة عبر iframe من `https://phet.colorado.edu/sims/html/...`

---

## المراحل

### المرحلة 1: البنية التحتية (قاعدة البيانات)

إنشاء جدول `lesson_simulations`:

```text
lesson_simulations
├── id (uuid, PK)
├── lesson_id (uuid, FK → lessons)
├── title (text) — اسم التجربة بالعربي
├── description (text, nullable) — وصف مختصر
├── phet_url (text) — رابط المحاكاة
├── thumbnail_url (text, nullable) — صورة مصغرة
├── sort_order (integer, default 0)
├── created_at (timestamptz)
```

سياسات RLS:
- SELECT: الجميع (public)
- ALL: الأدمن فقط

### المرحلة 2: واجهة عرض التجارب في صفحة الدرس

- إضافة تبويب جديد "التجارب 🔬" في `LessonPage.tsx` يظهر فقط عند وجود تجارب مرتبطة بالدرس
- إنشاء مكون `LabSimulation.tsx`:
  - عرض قائمة التجارب المتاحة للدرس (إذا أكثر من واحدة)
  - تضمين المحاكاة عبر iframe بملء الشاشة
  - زر فتح في نافذة جديدة وزر ملء الشاشة
  - تعليمات استخدام التجربة بالعربي

### المرحلة 3: إدارة التجارب في لوحة تحكم الأدمن

- إضافة صفحة `AdminSimulations.tsx` أو دمجها في صفحة إدارة الدروس
- واجهة لإضافة/تعديل/حذف التجارب المرتبطة بكل درس
- فلاتر متتالية (مرحلة → مادة → درس) كما في النمط الحالي
- إمكانية لصق رابط PhET مباشرة

### المرحلة 4: تعبئة البيانات

- إضافة روابط PhET الفعلية لدروس العلوم والفيزياء والكيمياء للمرحلة الثانوية
- أمثلة على محاكيات متاحة:
  - فيزياء: الحركة، القوى، الدوائر الكهربائية، الموجات
  - كيمياء: التفاعلات، تركيب الذرة، المحاليل
  - أحياء: الانتخاب الطبيعي، الوراثة

## التفاصيل التقنية

**التعديلات على الملفات الحالية:**
- `LessonPage.tsx`: إضافة query لجلب التجارب + تبويب "التجارب" الديناميكي
- `App.tsx`: إضافة route لصفحة إدارة التجارب (اختياري)

**الملفات الجديدة:**
- `src/components/LabSimulation.tsx` — مكون عرض المحاكاة
- `src/pages/admin/AdminSimulations.tsx` — إدارة التجارب (اختياري)

**تضمين PhET:**
```text
<iframe src="https://phet.colorado.edu/sims/html/circuit-construction-kit-dc/latest/circuit-construction-kit-dc_ar.html"
        allowfullscreen />
```
ملاحظة: معظم محاكيات PhET تدعم اللغة العربية بإضافة `_ar` في الرابط.

---

## ترتيب التنفيذ المقترح

1. إنشاء جدول `lesson_simulations` + RLS
2. بناء مكون `LabSimulation.tsx`
3. دمج التبويب في `LessonPage.tsx`
4. إضافة واجهة إدارة التجارب للأدمن
5. تعبئة بيانات تجريبية (روابط PhET حقيقية)
